# Diagrams

Editable architecture and workflow diagrams.

## Recommended Tools

- Draw.io / diagrams.net
- Mermaid (text-based)
- Lucidchart
- PlantUML

## What to Include

- System architecture diagrams
- Data flow diagrams
- Sequence diagrams
- Component interaction diagrams
- Deployment diagrams

## File Formats

- `.drawio` - Draw.io source files
- `.mmd` - Mermaid markdown
- `.puml` - PlantUML source
- `.png` / `.svg` - Exported images
