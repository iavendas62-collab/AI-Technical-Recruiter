# Images

Screenshots and visual assets for documentation.

## Guidelines

- Use fictional data only in screenshots
- No real personal information
- No sensitive credentials or API keys
- Preferred format: PNG or JPG
- Maximum file size: 2MB per image

## Organization

Place screenshots in appropriate subdirectories:
- `architecture/` - System diagrams and architecture
- `ui/` - User interface screenshots
- `workflows/` - n8n workflow screenshots
- `results/` - Output examples and results
